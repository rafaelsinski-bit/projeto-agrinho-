### O futuro não é herdado. É cultivade 
Este projeto consiste no desenvolvimento de um site informativo sobre sustentabilidade e preservação do futuro, criado na plataforma Base44. O objetivo principal do site é conscientizar os usuários sobre práticas sustentáveis no dia a dia e mostrar como pequenas ações podem contribuir para um futuro mais equilibrado e responsável para o meio ambiente.

### 🎯 Objetivo do site

O site tem como finalidade:

* Apresentar informações claras e acessíveis sobre sustentabilidade;

* Incentivar hábitos ecológicos no cotidiano;

* Promover a conscientização ambiental entre jovens e adultos;

* Servir como uma plataforma educativa e interativa sobre preservação do meio ambiente.

### ⚙️ Recursos técnicos

O projeto foi desenvolvido com foco em simplicidade, organização e boa experiência do usuário. Entre os principais recursos implementados estão:

* Design responsivo: adaptação do layout para diferentes dispositivos, como celulares, tablets e computadores;

* Navegação intuitiva: menus e seções organizadas para facilitar o acesso às informações;

* Conteúdo dinâmico: páginas estruturadas para apresentar textos, imagens e dicas sobre sustentabilidade de forma atrativa;

* Interface moderna: utilização de cores, tipografia e elementos visuais alinhados ao tema ambiental.

### 🛠️ Tecnologia utilizada

O site foi produzido utilizando a plataforma Base44, que permite a criação de aplicações web de forma prática e eficiente. Além disso, o projeto utiliza tecnologias web padrão, como:

* HTML – estruturação das páginas;

* CSS – estilização e design responsivo;

* JavaScript – interatividade e funcionalidades da interface.

A combinação dessas tecnologias possibilita um site leve, funcional e de fácil manutenção.

### 🌱 Considerações finais

Este projeto busca unir tecnologia e conscientização ambiental, mostrando como a web pode ser utilizada como ferramenta de educação e transformação social. O desenvolvimento no Base44 permitiu criar uma solução prática e moderna para compartilhar informações importantes sobre sustentabilidade e preservação do futuro.
